# 🏡 Smart Village Portal – Rajaqpur

A full-stack web application designed to digitize village services, improve communication between citizens and administration, and provide a modern digital experience for rural communities.

---

## 🚀 Features

### 👤 User Features
- Register & Login system (Authentication)
- Submit complaints with category & description
- Upload image with complaint (optional)
- View personal complaint history
- Edit & delete complaints
- Dark Mode toggle 🌙
- Profile management

### 📢 Announcement System
- View latest village announcements
- Real-time updates from admin panel

### 🖼️ Gallery & Heritage
- Village image gallery with popup preview
- Heritage section showcasing historical places

### 🛠️ Admin Features
- Admin dashboard
- Manage complaints (update status: Pending / Resolved)
- Add announcements

## ⚙️ Installation & Setup

1. Install XAMPP / WAMP
2. Start Apache & MySQL
3. Copy project folder into:

htdocs/

4. Open phpMyAdmin
5. Create a database (e.g., `smart_village`)
6. Import your SQL file (tables: users, complaints, announcements)
7. Update database connection in:

includes/db.php

8. Run project in browser:

http://localhost/smart_village/
